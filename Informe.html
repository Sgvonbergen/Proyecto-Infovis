Situación de dominio

Contexto de la situación y la naturaleza de los datos

Respuesta:

El conjunto de datos "World Population Insights 1970-2022" es un conjunto de datos demográficos que proporciona información sobre la población de diferentes países desde 1970 hasta 2022. Los datos incluyen información como la población total, la tasa de crecimiento de la población, la densidad de población, y otros indicadores demográficos. Este conjunto de datos puede ser útil para analizar tendencias de población y cambios demográficos a lo largo del tiempo.

Fuente original de los datos

Respuesta:

Los datos mencionados provienen de tres fuentes principales.

      El primer conjunto de datos es de Kaggle, titulado "World Population Insights 1970-2022". Este conjunto de datos proporciona información detallada sobre la población mundial desde 1970 hasta 2022, incluyendo datos demográficos y estadísticas de población para diferentes países.
      El segundo conjunto de datos es una enumeración CCA3 para los países, que es un estándar de tres letras para la codificación de nombres de países. Este estándar se utiliza para proporcionar una forma uniforme y reconocida internacionalmente de referirse a los países.
      El tercer conjunto de datos, también de Kaggle, proporciona información de geolocalización para los países, incluyendo las coordenadas de latitud y longitud. Este conjunto de datos se utiliza para cruzar los datos de población con las coordenadas geográficas de cada país.
      World Population Insights 1970-2022
      Geolocation
      CCA3

Abstracción de tareas

Tarea 1 (acción + objetivo) + justificación según el dominio presentado

Respuesta:

Mostrar la población de todos los paises del mundo. El objetivo es mostrar la distribución de la población en los distintos paises y su posición geografica

Tarea 2 (acción + objetivo) + justificación según el dominio presentado

Respuesta:

Mostrar y Comparar la población de paises seleccionados a lo largo del tiempo. El objetivo es permitir ver la tendencia de crecimiento en distintos paises y comparar la cantidad y cambio de población de distintos paises

Tarea 3 (acción + objetivo) + justificación según el dominio presentado

Respuesta:

Mostrar y comparar la población de la ciudad mas grande en los paises seleccionados. El objetivo es descubrir la población de las ciudades mas pobladas de ciertos paises y compararlas entre si.

Codificación visual e interacciones

Decisiones de diseño relacionadas a la primera tarea visual

Respuesta:

  1. Codificaciones visuales:
        2. Posición: Las coordenadas de latitud y longitud de cada país se utilizan para determinar la posición de los círculos en el mapa.
        Color: Se utiliza una escala de color logarítmica para representar la población de cada país. Los colores varían desde "#F3FF00" (amarillo) para las poblaciones más pequeñas hasta "#FF0000" (rojo) para las poblaciones más grandes.
        Tamaño: El tamaño de cada círculo se determina por la población del país. Se utiliza una escala logarítmica para determinar el radio de cada círculo en función de la población.
        Forma: Todos los datos se representan como círculos en el mapa.
  2. Interacciones:
        3. Zoom y panorámica: Los usuarios pueden hacer zoom y panorámica en el mapa. Cuando se hace zoom, el tamaño de los círculos se ajusta para mantener la proporción correcta.
        Clic: Al hacer clic en un círculo, se añade el país correspondiente a una lista de países seleccionados.
        Tooltip: Cuando el usuario pasa el cursor sobre un círculo, se muestra un tooltip con el nombre del país y su población.
        Actualización automática: Hay un intervalo establecido que actualiza la visualización cada segundo. Durante cada actualización, se selecciona un año al azar de la lista de años y se actualizan los círculos para reflejar los datos de población de ese año.
        Control de audio: Hay funciones para reproducir, pausar y detener una canción. La canción se reproduce mientras la visualización está activa.
        Selección de año: Hay una lista desplegable que permite al usuario seleccionar un año específico. Cuando se selecciona un año, se actualizan los círculos para reflejar los datos de población de ese año.

Decisiones de diseño relacionadas a la segunda tarea visual

Respuesta:

Las marcas elegidas en esta visualización son:

      Linea de conexion: Se utiliza una linea para conectar los puntos que representan la población en cada año
      Punto: Se utiliza un punto para marcar la posición del dato de población en el grafico
      Punto: Se utiliza un punto al lado del nombre de cada país para relacionarlo con el color y permitir eliminarlo de la lista al hacerle click
      Texto: Se utiliza texto para comunicar el nombre del pais representado en el grafico
      Cruz: Se utiliza un simbolo de cruz cuando se hace hover en un punto de la leyenda para representar que puede ser eliminado

Se utilizan dos canales en esta visualizacion:

      Posición: La posición de los puntos codifica el valor de la población en el pais en un año particular
      Color: El color de los puntos/linea en el grafico y del punto en la leyenda sirven como identificador para asociar la linea a un país

Existen dos glifos:

      Linea del grafico: incluye puntos, linea y color para representar el valor de la población de un pais
      Linea de la leyenda: incluye un punto, color y texto para asociar un color al cada pais

De acuerdo a los idioms utilizados:

      Idiom Visual: se decidio usar lineas para unir los distintos puntos porque queremos mostrar la tendencia de cambio de población a lo largo del tiempo. Los puntos se utilizan para marcar el dato en particular
      Idiom interaccion: Se decidio permitir la interacción con el grafico permitiendo eliminar uno de los paises seleccionados al hacer click en

Decisiones de diseño relacionadas a la tercera tarea visual

Respuesta:

  1. Codificaciones visuales:
        2. Posición: La posición en el eje Y de cada barra se determina por el nombre del país. La posición en el eje X de cada barra se determina por la población del país.
        Color: Cada barra tiene un color que se determina por el nombre del país. Se utiliza una escala de color ordinal para asignar un color único a cada país.
        Tamaño: El tamaño (ancho) de cada barra se determina por la población del país. Se utiliza una escala lineal para determinar el ancho de cada barra en función de la población.
  2. Interacciones:
        3. Hover: Cuando el usuario pasa el cursor sobre una barra, el grosor del borde de la barra aumenta y el color del borde se vuelve negro.
        Clic: Al hacer clic en una barra, se añade el país correspondiente a una lista de países seleccionados.
        Tooltip: Cuando el usuario pasa el cursor sobre una barra, se muestra un tooltip con la población del país.
        Actualización de datos: Cuando se llama a la función mostrargrafico, se actualizan los datos del gráfico de barras. Las barras existentes se actualizan para reflejar los nuevos datos, y las barras que ya no están en los datos se eliminan. Las nuevas barras se añaden para los nuevos datos.
        Selección de datos: Al principio del código, se seleccionan los 8 países con la mayor población para el año especificado. Estos datos se utilizan para generar el gráfico de barras.
